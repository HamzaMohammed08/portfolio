# Portfolio Website - Complete Implementation

## Overview
A modern, professional portfolio website for Hamza Muhammed (Software & AI Engineer) built with React, featuring a clean purple/lime color scheme inspired by the reference design.

## Pages Implemented

### 1. **Home** (`/`)
- Split hero section with purple (left) and lime (right) backgrounds
- Large, bold typography: "Software Engineer."
- Profile card with avatar circle and tech stack pills
- Call-to-action button to view projects
- Geometric decorative elements (dots pattern, zigzag)

### 2. **My Work** (`/projects`)
- Grid layout showcasing 6 projects
- Each project card features:
  - Color-coded header (alternating purple/lime)
  - Project category and title
  - Description and tech tags
  - Links to live demo and code
- Hover animations for enhanced interactivity

### 3. **My Shelf** (`/about`)
- Personal story and background
- Education timeline (Willowmoore High School, Eduvos)
- Quick facts sidebar with icons
- Interests & passions section
- Tools & technologies organized by category

### 4. **My Résumé** (`/resume`)
- Professional experience section
- Education history
- Comprehensive skills breakdown
- Download PDF button
- Clean, print-friendly layout

### 5. **Contact** (Accessible via side panel)
- Contact form with name, email, and message fields
- Email and social media cards
- Availability status indicator
- Form validation and styling

## Navigation
- **Fixed navbar** with "hamza." logo
- **Hamburger menu** that opens a side panel overlay
- Side panel includes:
  - Large navigation links (My Work, My Shelf, My Résumé)
  - Contact information (email, telegram)
  - Social media icons (Twitter, Github, LinkedIn)
  - Smooth slide-in animation

## Design Features
- **Color Scheme**: Deep purple (#5b21b6) and lime green (#d9f99d)
- **Typography**: Space Grotesk (headings) and Outfit (body)
- **Animations**: Framer Motion for smooth page transitions and hover effects
- **Responsive**: Mobile-friendly with adaptive layouts
- **Geometric Elements**: Dot patterns and zigzag decorations

## Tech Stack
- React 18
- React Router DOM (routing)
- Framer Motion (animations)
- Lucide React (icons)
- CSS Variables (theming)

## Key Optimizations
1. **Performance**: Lazy loading and optimized animations
2. **Accessibility**: Semantic HTML and proper contrast ratios
3. **SEO**: Meta tags and proper heading hierarchy
4. **UX**: Smooth transitions and intuitive navigation
